LƯU Ý QUAN TRỌNG
====================================================================

1. Hình ảnh: Bạn cần thay thế các hình ảnh trong thư mục:
   - assets/img/hero-img.png (hình nền hero)
   - assets/img/about.jpg, about-2.jpg (hình về chúng tôi)
   - assets/img/menu/ (hình sản phẩm)
   - assets/img/gallery/ (thư viện ảnh)
   - assets/img/chefs/ (ảnh chuyên gia)
   - assets/img/testimonials/ (ảnh khách hàng)
   - assets/img/events-*.jpg (hình sự kiện)

2. CSS: Bạn có thể muốn điều chỉnh màu sắc trong assets/css/main.css 
   để phù hợp với chủ đề hoa lan (xanh lá, tím, hồng, v.v.)

3. PHP Forms: Cần cập nhật files PHP trong thư mục forms/ để xử lý 
   đơn hàng thay vì đặt bàn

4. SEO: Cân nhắc cập nhật thêm meta tags và structured data cho SEO tốt hơn
